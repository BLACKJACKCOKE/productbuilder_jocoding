로또 번호 자동 생성 무료 서비스

LOTTO PICKER는 매주 로또 번호를 쉽게 생성하고 당첨 결과를 비교·분석할 수 있는 무료 서비스입니다. 1~45번 중 6개 번호를 랜덤으로 추천해 드리며, 주차별로 저장하여 당첨 번호와 자동으로 비교해 드립니다.

수집된 당첨 번호를 바탕으로 번호별 출현 빈도를 통계로 시각화해 드려, 나만의 번호 선택 전략을 세우는 데 도움이 됩니다.

📖 이용 방법

1

추첨 주차 선택

날짜 선택기에서 이번 주 토요일 날짜를 선택하세요. 기본값으로 가장 가까운 토요일이 자동 설정됩니다.

2

번호 생성

"번호 추가" 버튼을 눌러 1~45번 중 6개의 랜덤 번호를 생성합니다. 주차당 최대 5세트까지 추가할 수 있습니다.

3

주차 저장

"이 주차 저장" 버튼을 눌러 번호를 저장합니다. 저장된 번호는 "주차 관리" 탭에서 언제든 확인 가능합니다.

4

당첨 번호 입력 및 비교

추첨 후 "주차 관리" 탭에서 해당 주차를 선택하고 실제 당첨 번호를 입력하면 등수가 자동으로 계산됩니다.

5

통계 분석

"통계" 탭에서 누적된 당첨 번호 데이터를 기반으로 각 번호의 출현 빈도를 확인할 수 있습니다.

🎱 로또 당첨 확률 및 등수 기준

🏆 1등 — 6개 일치

본인 번호 6개가 당첨 번호 6개와 모두 일치. 당첨 확률 약 1/8,145,060

🥈 2등 — 5개 + 보너스

5개 번호 일치 + 보너스 번호 일치. 확률 약 1/1,357,510

본인 번호 중 5개가 당첨 번호와 일치. 확률 약 1/35,724

본인 번호 중 4개가 당첨 번호와 일치. 고정 상금 5만원

5등 — 3개 일치

본인 번호 중 3개가 당첨 번호와 일치. 고정 상금 5천원

❓ 자주 묻는 질문

Q. 번호 생성 알고리즘은 어떻게 되나요?
브라우저의 Math.random()을 활용한 피셔-예이츠 셔플 알고리즘으로 1~45 중 7개 번호를 추출합니다. 앞 6개가 메인 번호, 7번째가 보너스 번호로 배정됩니다. 완전한 랜덤으로 특정 번호에 유리함이 없습니다.
Q. 저장된 번호는 어디에 보관되나요?
모든 데이터는 서버로 전송되지 않으며, 현재 브라우저 세션 내에서만 유지됩니다. 브라우저를 새로고침하거나 닫으면 초기화됩니다. 개인 정보가 외부로 저장되지 않습니다.
Q. 통계를 보면 특정 번호가 더 자주 나오나요?
통계는 사용자가 직접 입력한 당첨 번호를 기반으로 집계됩니다. 로또는 매 회차 독립적인 사건이므로 과거 통계가 미래 결과에 영향을 주지 않습니다. 통계는 참고 자료로만 활용하세요.
Q. 모바일에서도 사용할 수 있나요?
네, 반응형 디자인으로 스마트폰, 태블릿, PC 모든 환경에서 최적화된 화면을 제공합니다.
Q. 이 서비스는 유료인가요?
완전 무료입니다. 회원가입, 로그인 없이 누구나 즉시 사용할 수 있습니다.
